# Lista de Exercicios da Unidade 3.2 do curso de Projeto de construção de algoritmos

Este repositório contém uma lista de exercícios desenvolvidos em **Python**, utilizando **estruturas de repetição (FOR e WHILE)**.

## Como executar os exercícios

1. Certifique-se de ter o **Python** instalado em sua máquina.
2. Clone o repositório ou baixe os arquivos.
3. Abra o terminal na pasta do projeto.
4. Execute o exercício desejado com o comando:

```
python Cursos/PCA/3.2/exercicios/nome_do_arquivo.py
```

Exemplo:
```
python Cursos/PCA/3.2/exercicios/exercicio1.py
```

Cada exercício é executado separadamente e solicitará dados do usuário pelo terminal.

## Lista de Exercícios

* **Exercício 1:** Exibição dos números de 1 até 10 utilizando a estrutura de repetição **for**.
* **Exercício 2:** Exibição dos números de 1 até 10 utilizando a estrutura de repetição **while**.
* **Exercício 3:** Simulação de um sistema de login que solicita usuário e senha até que os dados corretos sejam inseridos.
* **Exercício 4:** Simulação de login com número limitado de tentativas utilizando **for**, bloqueando o acesso após exceder o limite.
* **Exercício 5:** Soma de números inteiros positivos informados pelo usuário até que seja digitado o valor **0**.
* **Exercício 6:** Geração de uma tabela de multiplicação de 1 a 5 utilizando loops **for** aninhados.
* **Exercício 7:** Identificação e exibição de todos os números primos menores ou iguais a um número informado pelo usuário.
* **Exercício 8:** Geração dos **m** primeiros números da sequência de Fibonacci com base em um valor fornecido pelo usuário.
