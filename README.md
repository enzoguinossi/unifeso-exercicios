# 📚 Exercícios da Faculdade

Este repositório reúne exercícios, atividades e projetos desenvolvidos ao longo da graduação.
O objetivo é organizar o conteúdo por disciplina e unidade, facilitando o estudo, a revisão e o acompanhamento da evolução acadêmica.

---

## 🗂 Estrutura do Repositório

```text
Cursos/
├── PCA/
│   ├── 2.1-2.2/
│   └── 3.1/
├── RLM/
└── ERPA/
```

Cada pasta representa uma disciplina, e dentro dela estão organizadas as unidades ou listas de exercícios.
